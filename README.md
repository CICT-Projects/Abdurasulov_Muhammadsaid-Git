# 🚗 CRUD Приложение для управления автомобилями

Полностью функциональное веб-приложение для CRUD операций с автомобилями, построенное на **React** (фронтенд) и **.NET 10** (бэкенд).

## ✨ Функционал

- ✅ **Добавление машин** - создание новых записей об автомобилях
- ✅ **Просмотр машин** - отображение списка всех машин в красивой галерее
- ✅ **Редактирование машин** - изменение параметров существующих автомобилей
- ✅ **Удаление машин** - удаление записей об автомобилях
- ✅ **Живое обновление** - интерфейс синхронизируется с сервером в реальном времени
- ✅ **Отзывчивый дизайн** - работает на мобильных, планшетах и ПК
- ✅ **Валидация** - проверка корректности введённых данных
- ✅ **Красивый интерфейс** - современный дизайн с анимациями

## 🔧 Требования

- **.NET 10 SDK** или выше
- **Node.js 18+** и **npm**

## 🚀 Запуск приложения

### 1. Запуск бэкенда (.NET API)

```bash
cd TaskApi
dotnet run
```

API будет доступна по адресу: `https://localhost:5026`

### 2. Запуск фронтенда (React)

В новом терминале:

```bash
cd frontend
npm run dev
```

Приложение откроется по адресу: `http://localhost:5173`

## 📡 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|---------|
| GET | `/api/cars` | Получить все машины |
| GET | `/api/cars/{id}` | Получить машину по ID |
| POST | `/api/cars` | Создать новую машину |
| PUT | `/api/cars/{id}` | Обновить машину |
| DELETE | `/api/cars/{id}` | Удалить машину |

## 📝 Пример запроса (POST)

```json
{
  "brand": "Toyota",
  "model": "Camry",
  "year": 2023,
  "color": "Black"
}
```

## 🔐 CORS

CORS включен для всех методов и заголовков (настроено в `Program.cs`).

## 💾 Хранилище данных

На текущий момент машины хранятся в памяти приложения. При перезагрузке сервера данные будут потеряны.

Для постоянного хранения данных можно добавить:
- **Entity Framework Core + SQL Server**
- **MongoDB**
- **PostgreSQL**
