# 🚀 Быстрый старт

## За 2 минуты до первого запуска!

### Вариант 1️⃣: Автоматический запуск (Windows)

1. Убедитесь, что установлены:
   - [.NET 10 SDK](https://dotnet.microsoft.com/download)
   - [Node.js 18+](https://nodejs.org/)

2. Откройте терминал в папке `HomeTask`

3. Запустите:
   ```bash
   run.bat
   ```

4. Откроется:
   - Фронтенд: http://localhost:5173
   - Бэкенд API: https://localhost:5026

---

### Вариант 2️⃣: Ручной запуск

#### Шаг 1: Запустите бэкенд
```bash
cd TaskApi
dotnet run
```
Сервер запустится на `https://localhost:5026`

#### Шаг 2: В новом терминале запустите фронтенд
```bash
cd frontend
npm run dev
```
Приложение откроется на `http://localhost:5173`

---

## ✨ Что вы сможете делать?

### В веб-интерфейсе:

1. **Добавлять машины** 🚗
   - Заполните форму (марка, модель, год, цвет)
   - Нажмите "Добавить"

2. **Просматривать** 👀
   - Все машины отображаются в красивой галерее
   - Видны все параметры: марка, модель, год, цвет

3. **Редактировать** ✏️
   - Нажмите "Редактировать" на карточке
   - Измените нужные данные
   - Сохраните изменения

4. **Удалять** 🗑️
   - Нажмите "Удалить" на карточке
   - Подтвердите удаление

## 🧪 Тестирование API (опционально)

Используйте REST клиент (например, Postman или REST Client в VS Code):

```http
### Получить все машины
GET http://localhost:5026/api/cars

### Добавить машину
POST http://localhost:5026/api/cars
Content-Type: application/json

{
  "brand": "Toyota",
  "model": "Camry",
  "year": 2023,
  "color": "Black"
}

### Обновить машину
PUT http://localhost:5026/api/cars/1
Content-Type: application/json

{
  "brand": "Honda",
  "model": "Civic",
  "year": 2023,
  "color": "Blue"
}

### Удалить машину
DELETE http://localhost:5026/api/cars/1
```

Или просто используйте файл `TaskApi/CarsApi.http` в VS Code с расширением REST Client!

## 🆘 Если что-то не работает

### Проблема: "Порт 5026 уже используется"
```bash
netstat -ano | findstr :5026
taskkill /PID <PID> /F
```

### Проблема: "Node modules не установлены"
```bash
cd frontend
npm install
```

### Проблема: ".NET SDK не найден"
Установите [.NET 10 SDK](https://dotnet.microsoft.com/download)

### Проблема: Фронтенд не подключается к API
Убедитесь, что оба сервера запущены на правильных портах:
- Фронтенд: http://localhost:5173
- Бэкенд: https://localhost:5026
