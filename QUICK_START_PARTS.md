# 🚀 Быстрый старт

## Шаг 1: Запуск Backend (API)

```powershell
cd C:\Users\USER\Desktop\HomeTask\TaskApi
dotnet run
```

Backend запустится на: **http://localhost:5026**

## Шаг 2: Запуск Frontend (React)

В новом терминале:

```powershell
cd C:\Users\USER\Desktop\HomeTask\frontend
npm install
npm run dev
```

Frontend откроется на: **http://localhost:5173**

## Шаг 3: Открыть приложение

Откройте браузер и перейдите на:
```
http://localhost:5173
```

Вы увидите главную страницу с 5 секциями:
- 🚗 **Машины** - управление автомобилями
- ⚙️ **Моторы** - управление двигателями
- 🪟 **Окна** - управление стеклами
- 🛞 **Шины** - управление шинами
- 🔧 **Корпусы** - управление кузовами

## Что создано?

### 📦 Backend (Program.cs)

Добавлены модели для 4 запчастей:
1. **Motor** - Моторы (Мощность, Тип, Цена)
2. **Window** - Окна (Позиция, Тонировка, Цена)
3. **Tire** - Шины (Размер, Износ, Цена)
4. **Body** - Корпусы (Тип, Материал, Цена)

Созданы сервисы с полным CRUD функционалом для каждой запчасти.

### 🎨 Frontend (React)

Созданы компоненты:
- **Home.jsx** - Главная страница с красивой навигацией
- **Motors.jsx** - CRUD для моторов
- **Windows.jsx** - CRUD для окон
- **Tires.jsx** - CRUD для шин
- **Bodies.jsx** - CRUD для корпусов
- **App.jsx** - Маршрутизация между страницами

Все компоненты имеют стили и полностью функциональны.

## 🧪 Тестирование API

### Получить все моторы:
```bash
curl http://localhost:5026/api/motors
```

### Добавить новый мотор:
```bash
curl -X POST http://localhost:5026/api/motors `
  -H "Content-Type: application/json" `
  -d '{
    "name": "V12 Engine",
    "power": 600,
    "type": "Бензин",
    "price": 7000
  }'
```

### Обновить мотор (ID=1):
```bash
curl -X PUT http://localhost:5026/api/motors/1 `
  -H "Content-Type: application/json" `
  -d '{
    "name": "V8 Turbo Modified",
    "power": 500,
    "type": "Бензин",
    "price": 5500
  }'
```

### Удалить мотор (ID=1):
```bash
curl -X DELETE http://localhost:5026/api/motors/1
```

## 🐛 Решение проблем

### Port уже занят?
Измените порт в `Program.cs` или закройте приложение, использующее порт.

### npm install не работает?
```bash
npm install --force
```

### Нет подключения к API?
Убедитесь, что backend запущен на http://localhost:5026

## 📝 Примечания

- Данные сохраняются автоматически в JSON файлы
- Нет необходимости в БД (используются файлы)
- Все запчасти независимы друг от друга
- Машины остаются независимы от запчастей
