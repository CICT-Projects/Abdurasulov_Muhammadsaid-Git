- ✅ **Сервис CarService** - с методами CRUD (Create, Read, Update, Delete)
- ✅ **CarsController** - REST контроллер с 5 endpoints:
  - GET /api/cars - получить все машины
  - GET /api/cars/{id} - получить машину по ID
  - POST /api/cars - создать машину
  - PUT /api/cars/{id} - обновить машину
  - DELETE /api/cars/{id} - удалить машину
- ✅ **CORS** - настроен для работы с React приложением

## 🔍 Примеры использования

### Добавление машины:
1. Заполните поля формы
2. Нажмите кнопку "Добавить"
3. Машина появится в списке

### Редактирование машины:
1. Нажмите "Редактировать" на карточке
2. Форма заполнится данными машины
3. Измените нужные поля
4. Нажмите "Сохранить"

### Удаление машины:
1. Нажмите "Удалить" на карточке
2. Подтвердите удаление
3. Машина будет удалена
